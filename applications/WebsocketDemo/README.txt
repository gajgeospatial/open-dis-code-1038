Demo program for websockets, 3D content, and google maps.

The html and javascript files are in the content directory.
Run the projects, which starts a Jetty 9 web server, which is
also capable of being a server-side web sockets endpoint.
The server acts as a relay point for messages sent by the 
clients. The 3D scene in index.html displays some simple cubes
in a semi-random pattern. The map.html file displays a
version of the position of the cubes on a google maps
javascript display.
